Generative AI and labour productivity: a field experiment on coding

Leonardo Gambacorta, Han Qiu, Shuo Shan, Daniel Rees • 2024 • BIS Working Paper No. 1208
productivity coding software development novice workers experimental

Summary

  • Experiment from Sept - Nov 2023 at Ant Group involving ~1,200 coders, split between those with and without access to CodeFuse, a new coding assistance LLM. Primary measure = lines of code produced/week
  • Group with access to LLM saw a 53% increase in code production relative to control group. Of this, 11-18pp was due to coders using direct LLM outputs, and 36-43pp due to time saving, likely from the LLM helping coders overcome obstacles in coding.
  • Entry level and junior coders had 67% increase in code produced; senior coders no significant change.
  • Senior coders used AI 70% less than juniors, but acceptance rate for AI suggestions (15-20%) was the same across groups.

Abstract

Conducted a field experiment at Ant Group where a team of programmers received an AI coding assistant (CodeFuse) while a control group did not. Programmers with the AI tool wrote ~55% more code (lines) than those without. Productivity gains were statistically significant only for junior developers; senior programmers saw little change, apparently using the tool less often.
