Generative AI and labour productivity: a field experiment on coding

Leonardo Gambacorta, Han Qiu, Shuo Shan, Daniel Rees • 2024 • BIS Working Paper No. 1208
productivity coding software development novice workers experimental

Summary

Experiment at Ant Group involving ~1,200 coders. Group with access to LLM saw a 53% increase in code production relative to control group. Entry level and junior coders had 67% increase; senior coders no significant change.

Abstract

Conducted a field experiment at Ant Group where a team of programmers received an AI coding assistant (CodeFuse) while a control group did not. Programmers with the AI tool wrote ~55% more code (lines) than those without. Productivity gains were statistically significant only for junior developers; senior programmers saw little change, apparently using the tool less often.
